Pip安装以来报错

Posted on May 14, 2022
tl;dr:

使用pip安装以来出现了No space left on device错误

Collecting Pillow==8.2.0
  Using cached Pillow-8.2.0.tar.gz (47.9 MB)
ERROR: Could not install packages due to an OSError: [Errno 28] No space left on device

解决方法

转载：https://www.programminghunter.com/article/9003348245/

mkdir -p $HOME/tmp 
echo "export TMPDIR=$HOME/tmp" >> ~/.bashrc
source ~/.bashrc
